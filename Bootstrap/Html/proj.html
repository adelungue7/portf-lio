Projetos desenvolvidos neste ano em T.I


Jogo com a temática gravidade feito no Construct 3.0:

Neste projeto foi proposto a criação de um jogo com a temática de gravidade utilizando o Scratch, Construct 3.0 ou o MIT App Inventor 2.0, era necessário utilizar váriaveis e seus tipos de dados, estruturas condicionais, estruturas de repetição, entre outros. Também era preciso que o jogo tivesse condições de vitórias e derrotas, tela inicial, tela de créditos, power up, vidas, painel informativo e um sistema de pontuação. Além disso criamos uma documentação com todas as ferramentas que utilizamos e por fim gravamos um trailer para a divulgação do jogo.





Aplicativo em Java (Sistema de Visitantes)

Neste segundo projeto a ideia era criar um aplicativo utilizando a linguagem de programação Java no NetBeans, nele utilizamos diversas técnicas de programação que foram desenvolvidas no segundo bimestre, no Sistema criamos Tela de Login, Processo de Check-In, Coleta de Dados, Entrega de Brindes, Impressão de Etiquetas, entre outros. Neste projeto também foi criada uma documentação com todo o processo do projeto, contendo todas as ferramentas que foram utiizadas.





Sistema com Banco de Dados Integrado

Por fim temos o Sistema com Banco de Dados Integrado que foi realizado no terceiro bimestre deste ano, nele utilizamos a linguagem de programação Java no aplicativo NetBeans, também fizemos o Banco de Dados no MySQL WorkBench com diversas funcionalidades como operações CRUD (Criar, Ler, Atualizar e Deletar) por exemplo, também fizemos a documentação com todos os passos do projeto. Nele criamos um sistema de mobilidade urbana utilizando quatro dos municípios mais famosos de São Paulo, para que fosse mais fácil para essas pessoas identificarem o transporte público da região.
